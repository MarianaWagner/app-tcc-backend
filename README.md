Este projeto foi desenvolvido como parte do meu Trabalho de Conclusão de Curso (TCC) em Ciência da Computação, tendo como objetivo central a criação de uma solução mobile capaz de organizar e centralizar informações clínicas do paciente. A aplicação busca oferecer uma plataforma simples, intuitiva e segura para o armazenamento de exames e prescrições médicas, fortalecendo a autonomia do usuário e reduzindo a fragmentação de dados entre diferentes instituições de saúde.
